# 🏨 Restful-Booker Automation Suite

Proyecto de QA Automation diseñado para validar la API pública [Restful-Booker](https://restful-booker.herokuapp.com/). 
Este proyecto demuestra habilidades en **API Testing**, **automatización con Python** y **gestión de pruebas**.

## 🛠️ Tecnologías
* **Python** (Librerías: `requests`, `pytest`)
* **Postman** (Colecciones y validaciones manuales)
* **GitHub Actions** (Integración Continua - *Próximamente*)

## 📋 Estructura
* `/collections`: Colecciones de Postman (.json)
* `/tests`: Scripts de automatización en Python
* `TEST_CASES.md`: Detalle de los casos de prueba ejecutados
